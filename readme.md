# Orchard
A 2D platformer prototype in the works. Could become a game. Leaving public for code examples.

## Features
* OpenGL Renderer

## Dependencies
* C Runtime
* glext.h, wglext.h from Khronos

## Setup
1. Clone the repository. 
```sh
$ git clone https://github.com/colbyhall/orchard.git
```
2. Run build.bat
```sh
$ .\build.bat
```
